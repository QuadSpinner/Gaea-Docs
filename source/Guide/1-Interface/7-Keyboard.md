## File Commands
| Command          | Shortcut           |
| :---------------- | ------------------: |
| New              | `Ctrl + N`         |
| Open             | `Ctrl + O`         |
| Save             | `Ctrl + S`         |
| Save incremental | `Ctrl + Shift + S` |

## Graph Commands

| Command                    | Shortcut           |
| :-------------------------- | ------------------: |
| Refresh node               | `F5`               |
| Bypass node                | `B`                |
| Pin node preview           | `F`                |
| Pin node for color preview | `G`                |
| Mark node for save         | `F3`               |
| Delete node                | `Del`              |
| Rename node                | `F2`               |
| Build terrain              | `Ctrl + Shift + B` |
| MultiMix nodes             | `F8`               |
| Duplicate node             | `Ctrl + D`         |
| Quick search nodes         | `TAB`              |
| Zoom graph to 100%         | `Ctrl + 1`         |
| Zoom graph to fit all      | `Ctrl + 0`         |
