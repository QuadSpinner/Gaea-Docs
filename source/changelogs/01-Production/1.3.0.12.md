#### What's new in Gaea 1.3?

This our largest update to date, as well as our Long Term Support (LTS) version. 

For a glimpse of all the major features and improvements, see [the Gaea 1.3 release blog post](https://blog.quadspinner.com/gaea-1-3-now-available/).

## Changes in 1.3.0.12

- FIX: Thermal erosion reverted temporarily because of a new bug introduced in 1.3.0.11.
- FIX: Missing startup.tor file could prevent Gaea from working correctly.
- FIX: Watson did not clean startup.tor during clean installation.
- FIX: Layout was not updated when saving the workspace.
