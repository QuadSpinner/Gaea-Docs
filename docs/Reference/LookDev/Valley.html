Valley

The documentation is being updated for version 1.3. Some sections may be incomplete.

Valley is a specialized LookDev node for carving out valley structures in bulky terrains with large slopes.

Properties

Ridge Style
RoughThe ridges are rough. SmoothThe ridges have smooth slopes. SharpThe ridges have smooth slope with sharp edges.
Erosive Strength The amount of erosion in the valley structure.
Flow Width The width of flow in the valley
Depth The depth of the valley.
Base Level
Sediments Amount of sediment deposited in the valley.
Random
Deterministic (Slow)
SeedThe randomization pattern or seed for the node's process.
