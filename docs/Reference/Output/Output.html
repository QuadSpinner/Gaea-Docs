Output

The Output node gives you explicit options for saving a node's output to file. It is an alternative to the Mark for Export option that is available on all nodes. It is specifically useful for saving files to a predefined path and overwriting the output every time.

The output file will be overwritten on each build. Use the Build Manager instead to automatically stack and rename files.

The Output node is for backward compatibility only. We recommend using the Build Manager instead, unless you have specific needs.
