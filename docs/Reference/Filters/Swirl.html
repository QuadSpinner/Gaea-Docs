Swirl

Swirl creates a vortex warp on the terrain. It can be very useful for adding a fluid look to the terrain.

Properties

DegreesThe strength of the vortex. Negative values move the distortion clockwise; positive values move the distortion counter-clockwise.
ScaleThe size of the swirl.
