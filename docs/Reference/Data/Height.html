Height

Height

The Height node creates a selection mask within the height range specified.

Properties

MinThe minimum altitude to select.

MaxThe maximum altitude to select.

FalloffThe soft falloff at the maximum edge.

NormalizedBy default, the height selection happens between the base and maximum height set in the Terrain Definition. This may sometimes make it harder to select the exact height visually. When Normalized is enabled, the maximum height is the maximum height of the tallest peak in the current heightfield.
