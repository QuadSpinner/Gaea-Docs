Voronoi+

A more flexible version of Voronoi based on clustering that can be useful for advanced scenarios. While the Voronoi node uses a fixed set of points to generate the pattern, Voronoi+ uses randomized points that you can control.

Properties

IterationsThe number of generative passes. Higher numbers create more complexity, but may take a bit more time to generate.
Distribution
PoissonSeemingly random placement with a natural pattern. RandomFully random placement. GridGrid placement in strict columns and rows.
