Release Notes

IN THIS BUILD This is a stability update, with few new features. It addresses the mouse lag issue that had affected many users. With this bugfix in place, we will now resume adding new features every week.

See https://medium.com/quadspinner for details.

NEW: Detach viewport from main window. UPDATED: Origami results are now consistent at different resolutions. UPDATED: Startup performance increased. UPDATED: Slider precision increased to 4 decimals. FIXED: Layers were not updated when rearranged. FIXED: Extreme slowdown in Graph mode. FIXED: Wacom/Tablet drivers conflicted with mouse input.
