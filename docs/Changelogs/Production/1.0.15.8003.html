Gaea 1.0.15.8003

Released on 01 March 2019

Download 45.24MB

This is a stability update that fixes several issues reported by users.

[Improved] Portals can now be connected from context menu. [Improved] Portals are automatically named. [Improved] Portal tooltips show which portal it is connected to.

[Fixed] Reloading a file would destory terrain scale values. [Fixed] Auto Layout no longer zooms to extents. [Fixed] Save Current Preview would not save color files.
