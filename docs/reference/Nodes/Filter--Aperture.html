    The Aperture node expands or compacts every single feature on the terrain, similar to bokeh in a camera.

    SettingValuesDescription
    MethodChoose whether to expand or compact the visuals on the terrain.
    ExpandLorem
    ContractLorem
    Iterations0 - 20Number of aperture passes to perform on the terrain. Higher number of passes create pronounced effect at the expense of speed.
    KernelThe Kernel is the shape of the aperture. Changing the kernel can change how the features on the terrain are modified.
    DiskLarge, circular formation.
    DiamondSmall, diagonal squares.
    CrossSmall cross pattern. Can create predictable shape, use with masking for best results.
    OctLarge, octagonal formation.
    CornerLarge, diagonal squares.
    RingSmall, circular formation with steps/terraces.
    SobelRectangular stretches which retain corner shapes.
    RandomOn / OffWhen on, each iteration uses a randomly selected kernel.
    Seed0 - 128000The randomization pattern or seed for the node's process.
