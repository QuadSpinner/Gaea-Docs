title: Clamp

nClamp allows you to control the height of the terrain. You can use it squash the terrain, clip it at certain heights, or make it taller using the Extend operation.

| Setting       | Values      | Description |
| :------------ | :---------- | :---------- |
| **Min**       | 0.00 - 1.00 | Lorem       |
| **Max**       | 0.00 - 1.00 | Lorem       |
| **Operation** | Lorem       |
|               | *Clamp*     | Lorem       |
|               | *Clip*      | Lorem       |
|               | *Extend*    | Lorem       |
