title: Keyboard Shortcuts
---

| Command                    | Shortcut           |
| -------------------------- | ------------------ |
| **File Commands**        |
| New                        | `Ctrl + N`         |
| Open                       | `Ctrl + O`         |
| Save                       | `Ctrl + S`         |
| Save Incremental           | `Ctrl + Shift + S` |
| **Graph Commands**       |
| Refresh node               | `F5`               |
| Bypass node                | `B`                |
| Pin node preview           | `F`                |
| Pin node for color preview | `G`                |
| Mark node for save         | `L`                |
| Delete node                | `Del`              |
| Rename node                | `F2`               |
| Build Terrain              | `Ctrl + Shift + B` |
